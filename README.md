# Check Key Windows
## Check for Command
- กดปุ่ม `Windows` + `R`
- พิมพ์ `cmd` + `Enter`
- รันคำสั่งในหน้า Command Prompt ตามนี้
```
wmic path softwarelicensingservice get OA3xOriginalProductKey
```
- หากคอมพิวเตอร์ของเรามี OEM License ก็จะแสดง Product Key ขึ้นมา
